Я разработчик с практическим опытом создания серверных приложений, Telegram-ботов и веб-сервисов. Основной стек — **Node.js и TypeScript**, с фокусом на написании чистого, поддерживаемого и надёжного кода.

Работаю как с разработкой новых проектов, так и с развитием существующих систем: умею быстро разбираться в чужом коде, находить узкие места и аккуратно улучшать архитектуру без ломки работающего продукта.

Имею опыт разработки **Telegram-ботов, веб-API и веб-приложений**, интеграции внешних сервисов и построения стабильных backend-систем, рассчитанных на реальных пользователей и продакшн-нагрузку.

**Основные навыки:**

* Backend-разработка на **Node.js / TypeScript**
* Разработка **Telegram-ботов** и работа с Telegram Bot API

* Интеграция внешних API (AI-сервисы, платежные системы и др.)
* Работа с базами данных **PostgreSQL**
* Проектирование архитектуры и оптимизация производительности
* Логирование, мониторинг и отладка продакшн-сервисов

**Дополнительно:**

* опыт деплоя и работы с серверной инфраструктурой (VPS / VDS)
* настройка вебхуков, автоматических обновлений и стабильной работы сервисов
* понимание принципов масштабируемых систем и обработки высокой нагрузки

Мне интересны проекты, где можно не просто писать код, а **развивать продукт, улучшать архитектуру и создавать надёжные сервисы, которыми реально пользуются люди**.
